What the model learned gain-based feature importance, from training

How this works: the 70 restaurants — names, GPS coordinates, cuisines, prices, opening hours, and star ratings — are real, from the UCI "Restaurant & Consumer Data" set (San Luis Potosí, Mexico). A LightGBM LambdaMART model was trained offline on simulated search sessions over this real catalog (no public query-log data exists for it) and exported to this page, where a small JavaScript function walks its 120 trees directly — every ranking below is genuine model inference. Your search text is matched against each restaurant's real name and cuisine tags, distance is computed from real coordinates, and "open now" is read from each restaurant's real weekly hours for the time you pick above.
